# jq-customRadioCheck

A tiny jQuery plugin to customize `checkbox` and `radio` inputs.  

**Demo:** http://jsfiddle.net/elclanrs/BQp2F/  
**Support:** IE8+, Webkit, Firfox, Opera, iOS5+, Android 4.0+.  
**jQuery:** 1.7+  
**License:** MIT

## Usage:

```javascript
$('input[type=checkbox], input[type=radio]').customRadioCheck();
```
